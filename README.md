# advent-of-code-2025
My attempts at Advent of Code 2025 | https://adventofcode.com/2025

# How to run
1. Install [pnpm](https://pnpm.io/installation) if you don't have it already.
2. Run `pnpm install` to install dependencies.
3. Run `pnpm run day-{X}:part-{Y}` to run the solution for day X, part Y. For example, to run day 1, part 1, use `pnpm run day-1:part-1`.
